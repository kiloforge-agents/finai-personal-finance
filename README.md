# FinAI — Personal Finance AI Assistant

A high-converting landing page for FinAI, an AI-powered personal finance assistant with real-time in-store integration.

## Stack
- Next.js 16 (App Router)
- Tailwind CSS v4
- Lucide React icons
- PostHog analytics

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the page.
